SendTrace - Shipments is designed to provide information to your shippers and consignees, technology solution for shipments and logistics operations. 
It has core features to manage shipments in easy way, printing of shipment's documents, track shipments and notify your customers about their shipment status.

== Core Plugin Features ==

* Tracking Form - using shortcode [sendtrace_form]
* Manage Shipments
* Shipment Settings
* Address Book
* Email Notification
* Support Barcode
* PDF Invoice
* PDF Waybill
* Generate Report
* Multiple Packages

== Installation ==

1. Click Install button and Activate or Upload `sendtrace-shipments.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= Configuration =

1. Create page and insert [sendtrace_form] for tracking form.
2. Configure settings for your shipments and email notification.
4. Enjoy Managing of your shipments.
